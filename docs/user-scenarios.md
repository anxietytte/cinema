# Пользовательские сценарии

## Сценарий 1: Регистрация гостя

**Актор:** Гость

**Действия:**
1. Гость открывает главную страницу и нажимает "Регистрация"
2. Заполняет поля: email, пароль, имя
3. Нажимает "Зарегистрироваться"
4. Аккаунт активирован, выполнен автоматический вход

**Ограничения:**
- Пароль: минимум 8 символов, включая цифру
- Email должен быть валидным

## Сценарий 2: Работа пользователя

**Актор:** Авторизованный пользователь

**Действия:**
1. Пользователь входит в личный кабинет
2. Выбирает нужную функцию
3. Заполняет данные и нажимает подтверждение
4. Система проверяет данные и сохраняет их
5. Пользователь видит сообщение об успехе

**Ограничения:**
- Сессия истекает через 30 минут бездействия
- Время выполнения действия — до 5 минут

## Сценарий 3: Управление пользователями (администратор)

**Актор:** Администратор

**Действия:**
1. Администратор входит в систему
2. Переходит в раздел "Управление пользователями"
3. Находит нужного пользователя через поиск
4. Выбирает действие: блокировка, смена роли или удаление
5. Подтверждает действие
6. Система выполняет действие и записывает его в лог

**Ограничения:**
- Администратор не может заблокировать самого себя
- Все действия логируются

**Бизнес-ограничения:**
