# Карта навигации

## Маршрут гостя

Главная -> Регистрация -> Заполнение формы -> Вход -> Личный кабинет

Главная -> Вход -> Личный кабинет

Главная -> О проекте

## Маршрут авторизованного пользователя

Личный кабинет -> Профиль -> Редактирование профиля -> Личный кабинет

Личный кабинет -> Основная рабочая страница -> Добавление/редактирование данных -> Личный кабинет

Личный кабинет -> История действий

Личный кабинет -> Выход

## Маршрут администратора

Вход -> Панель управления -> Управление пользователями -> Карточка пользователя -> Панель управления

Вход -> Панель управления -> Общие настройки
